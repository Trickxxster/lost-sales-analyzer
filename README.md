# Анализ неудовлетворённого спроса

Это приложение на **Streamlit** позволяет оценить упущенную выгоду из-за дефицита товара на основе ежедневных данных о продажах и остатках.

## Как это работает

1. Вы загружаете Excel-файл с данными за месяц (или любой период).
2. Приложение вычисляет:
   - Остаток на начало дня (из предыдущего дня).
   - Поступления (по балансовому уравнению).
   - Оценку спроса (выбранным методом).
   - Дефицит (если остаток на начало < оценка спроса).
   - Упущенную выгоду (дефицит × цена).
3. Выводит графики и сводные метрики.

## Формат входных данных

Excel-файл должен содержать **обязательные** колонки:

| Колонка | Описание |
|---------|----------|
| `Дата` | Дата в формате, который распознает pandas (например, `01.04.2026`). |
| `Продано штук` | Количество проданных единиц за день (если продаж не было, ставьте 0 или оставляйте пустым). |
| `Остаток на конец периода, шт` | Остаток на конец дня. |
| `Стоимость 1 шт в рублях` | Цена за единицу (может меняться по дням). |
