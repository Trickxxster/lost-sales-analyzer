# Анализ неудовлетворённого спроса

Приложение для оценки упущенной выгоды из-за отсутствия товара на складе.

## Как работает
1. Загрузите Excel-файл со структурой: каждый день – отдельный блок, начинающийся со строки "Номенклатура".
2. Приложение извлекает продажи и остатки по городам и товарам.
3. Рассчитывает средний дневной спрос по дням, когда товар был в наличии.
4. Для дней с нулевым остатком считает дефицит = средний спрос.
5. Выводит графики, тепловую карту и детальную таблицу.

## Требования
- Python 3.12
- Установка зависимостей: `pip install -r requirements.txt`
- Запуск: `streamlit run app.py`

## Структура входного файла
- Каждый день начинается со строки, где в первом столбце "Номенклатура".
- Следующая строка – названия городов.
- Затем строка с подзаголовками (Класс шт, Количество, ...).
- Далее строки с товарами и данными по городам.
- Блок завершается строкой "Итого".

## Выходные данные
- Общий дефицит в штуках.
- График дефицита по дням.
- Детальный график остатков и дефицита для выбранного товара.
- Тепловая карта дефицита по товарам и городам.
- Возможность скачать результат в Excel.
